Uniform cells with a simple icon and skill name. More visual and inventory-like.

Headline on the left, three ruled skill columns on the right. Closest to the current Swiss/minimal feel.

Core stack + supporting list

A short row of primary skills, then a compact supporting list underneath for hierarchy.

Bento / mosaic map

Asymmetric tiles by domain — Cloud, Backend, Frontend, Data, Observability. Strongest visual identity.
